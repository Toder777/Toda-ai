# Toda-ai
Toda AI Agent — интеллектуальный AI-агент на LangGraph и DeepSeek. Управляет файлами (создание, чтение, запись), ищет в интернете, выполняет команды CMD, эмулирует клавиши, сохраняет заметки и погоду. Имеет долговременную память, мульти-чаты и REST API. Работает локально, бесплатно, без API-ключей. находиться в стадии разработки.

## 📦 Установка

### Требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Шаг 1: Скачивание проекта

Клонируйте репозиторий:
```bash
git clone https://github.com/Toder777/Toda-ai.git
cd toda-ai-agent
```

### Шаг 2: Установка зависимостей

найдите файл `requirements.txt` со следующим содержимым:

```txt
opendeep>=0.1.0
langgraph>=1.2.0
langchain>=1.3.0
langchain-core>=1.4.0
fastapi>=0.136.0
uvicorn>=0.49.0
requests>=2.32.0
pyautogui>=0.9.54
pydantic>=2.13.0
```

Затем выполните команду:
```bash
pip install -r requirements.txt
```

Или установите зависимости вручную:

```
pip install opendeep langgraph langchain fastapi uvicorn requests pyautogui
```

### Шаг 3: Получение токена DeepSeek

1. Перейдите на [chat.deepseek.com](https://chat.deepseek.com) и войдите в аккаунт
2. Нажмите клавишу `F12` для открытия инструментов разработчика
3. Перейдите на вкладку **Application** (Хранилище)
4. В левом меню выберите **Local Storage** → `https://chat.deepseek.com`
5. Найдите в списке ключ `userToken` и скопируйте его значение
6. Откройте файл `backend.py` и вставьте токен в переменную `DEEPSEEK_TOKEN`:

```
DEEPSEEK_TOKEN = "ваш_токен_здесь"
```

### Шаг 4: Запуск сервера
выполните в консоли команду:
```bash
python backend.py
```
Сервер запустится на `http://127.0.0.1:8000`

### Шаг 5: Открытие интерфейса

- **Главная страница:** http://127.0.0.1:8000
- **Чат:** http://127.0.0.1:8000/chat.html

## 💬 Использование

### Примеры запросов

| Запрос                                     | Что произойдёт                      |
|--------------------------------------------|-------------------------------------|
| `создай файл todo.txt с планом на сегодня` | Создаст файл с указанным содержимым |
| `прочитай файл backend.py`                 | Покажет содержимое файла            |
| `найди в интернете последние новости ИИ`   | Выполнит поиск через DuckDuckGo     |
| `какая погода в Москве`                    | Покажет текущую погоду              |
| `запомни: купить хлеб и молоко`            | Сохранит заметку в notes.txt        |
| `выполни команду dir /w`                   | Покажет содержимое папки            |
| `нажми клавиши CTRL+C`                     | Эмулирует нажатие клавиш            |
| `напомни про встречу в 15:00`              | Создаст напоминание                 |


### Управление чатами

- **Создать новый чат:** кнопка "➕ Новый чат"
- **Переименовать чат:** двойной клик по названию или клик по названию в шапке
- **Удалить чат:** навести на чат в боковой панели → нажать 🗑️
- **Прикрепить файл:** кнопка 📎 → выбрать файл → отправить на анализ

## 📡 API Endpoints

### POST `/chat`

Отправить сообщение агенту.

**Тело запроса:**

```json
{
  "message": "текст сообщения",
  "thread_id": "идентификатор_чата",
  "file": {
    "name": "имя_файла",
    "content": "содержимое"
  }
}
```

**Ответ:**

```json
{
  "response": "ответ агента"
}
```

### GET `/download/{filename}`

Скачать файлы проекта - для index  страницы (главная страница)

## ⚙️ Настройка

### Изменение порта

В файле `backend.py` последние строки:
```python
if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="127.0.0.1", port=8000)  # замените 8000 на нужный порт
```

### Смена модели DeepSeek

В `backend.py` измените `MODEL_NAME`:

```python
MODEL_NAME = "deepseek-chat"        # обычная модель
# MODEL_NAME = "deepseek-reasoner"  # модель с рассуждениями
```

### Очистка истории

Удалите файл `conversation_history.json` — все диалоги будут сброшены.

## 🔧 Устранение проблем

### Ошибка `Address already in use`

Порт 8000 занят другим процессом:

# Найти процесс
```bash
netstat -ano | findstr :8000

# Убить процесс (замените PID на число из предыдущей команды)
taskkill /PID 12345 /F
```

### Ошибка `DEEPSEEK_TOKEN` не задан

Получите новый токен из браузера (F12 → Application → Local Storage → `userToken`)

### Ошибка импорта `opendeep`

```bash
pip uninstall opendeep
pip install opendeep
```

### Агент не выполняет команды

Убедитесь, что запрос соответствует формату. Попробуйте явную команду:

- `создай файл test.txt`
- `выполни команду dir`

### Ошибка 530 при использовании cloudflared

Убедитесь, что сервер Toda запущен:

```bash
python backend.py
```

Затем в другом окне терминала запустите туннель.
