Toda AI Agent
🤖 LangGraph 🧠 DeepSeek 📁 Файловая система 🔍 Веб-поиск 💾 Долговременная память
Интеллектуальный AI-агент с полным контролем над файлами, поиском в интернете, выполнением команд и долговременной памятью. Работает полностью локально.

Версия 2.0 · Windows / macOS / Linux · Бесплатно

📁
Управление файлами
Создавайте, читайте, редактируйте и удаляйте файлы. Работа с текстовыми документами, JSON, кодом.
🔍
Веб-поиск
Ищите информацию в интернете через DuckDuckGo без API-ключей.
💾
Долговременная память
Агент помнит все диалоги даже после перезапуска сервера.
⌨️
Выполнение команд
Запускайте CMD-команды, скрипты, эмулируйте нажатия клавиш.
🌤️
Погода и заметки
Узнавайте погоду в любом городе, сохраняйте заметки в файл.
💬
Мульти-чаты
Создавайте несколько чатов, переименовывайте, удаляйте. Всё сохраняется.

✨ Примеры запросов

📄 создай файл todo.txt с планом на сегодня
Агент создаст файл и запишет в него ваш план.
🔍 найди в интернете последние новости ИИ
Поиск через DuckDuckGo, вернёт краткую выжимку из статей.
🌤️ какая погода в Санкт-Петербурге
Текущая температура, осадки, облачность.
📝 запомни: купить хлеб и молоко
Сохранит заметку в файл notes.txt с временной меткой.
💻 выполни команду dir /w
Покажет содержимое текущей директории в консоли.
⌨️ нажми клавиши CTRL+C
Эмулирует нажатие комбинации клавиш (требуется pyautogui).
